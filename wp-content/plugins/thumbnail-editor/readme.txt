Manually Crop and Resize thumbnail images that are uploaded in the Media section. 

== Description ==

When images are uploaded in the WordPress Media library section wordpress creates some thumbnails of that image. The size of these thumbnail images can be mentioned in the `Settings->Media` section. 

This plugin will allow you to update those thumbnail images manually. The thumbnail image edit interface that this plugin provides is simple and very easy to use.

The shortcodes and functions are provided in the image editor page of the plugin.

Image editor links can be found in WordPress Media Library.

Image files that can be edited are <strong>JPG, JPEG, GIF and PNG</strong>

== Installation ==


1. Upload `thumbnail-editor.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now go to `Media->Library` and in the files list you should see the "Crop Thumbnail" link. 
4. Click on the link to start editing the images.
